1. HTML создает кнопку для показа уведомления и контейнер для toasts. 2. CSS стилизует кнопку и уведомления, добавляя эффекты появления и исчезновения. 3. JavaScript добавляет обработчик события на кнопку. При нажатии на кнопку создается новый элемент уведомления, который добавляется в контейнер. Уведомление будет отображаться в течение 3 секунд, после чего плавно исчезнет.
